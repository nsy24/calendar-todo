# ファビコン・アプリアイコンの準備

ブラウザのタブやホーム画面に表示されるアイコンを SyncTask 用に差し替える場合の手順です。

## 推奨デザイン

- **モチーフ**: 「S」をモチーフにしたシンプルなアイコン
- **スタイル**: 角丸やグラデーションは控えめにし、視認性の高い単色または2色程度にすると、小さいサイズでも認識しやすいです

## 用意するファイル

次のパスに配置してください。

| 用途 | パス | 推奨サイズ |
|------|------|------------|
| ブラウザタブ用 | `public/favicon.ico` | 32×32 または 16×16, 48×48 をマルチアイコンで |
| PWA・Apple Touch | `public/icons/icon-192x192.png` | 192×192 px |
| PWA | `public/icons/icon-512x512.png` | 512×512 px |

## layout での参照（現状）

- `app/layout.tsx` の `<head>` に  
  `<link rel="apple-touch-icon" href="/icons/icon-192x192.png" />` が設定されています。
- ファビコン（タブ用）は Next.js のデフォルトで `app/icon.ico` または `public/favicon.ico` を自動参照します。  
  **「S」モチーフの ICO または PNG を `public/favicon.ico` に置くとタブに反映されます。**

## 作業の流れ（例）

1. デザインツールで「S」をモチーフに 512×512 の画像を作成し、PNG で書き出す。
2. 必要に応じて ICO 変換ツールで `favicon.ico`（マルチサイズ推奨）を生成し、`public/favicon.ico` に配置。
3. 192×192 と 512×512 の PNG を `public/icons/` に配置（既存ファイルを上書き）。
4. ブラウザでハードリロード（キャッシュ無効）して表示を確認。

以上で SyncTask 用のファビコン・アプリアイコンへの差し替えができます。
